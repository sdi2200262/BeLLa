### AI:

Claude 3.5 Sonnet is the best AI for implementing the components i want up until now. Im having trouble getting the OpenAI models to fully understand what i want (design wise) through the promt - however the reasoning models like o1-mini and o1-preview have seen some success. I think im still not in the very complex coding part of the project - maybe later on it will be even harder for the AI to cope since a lot of files will be connected to each other and hallucinations will become more common. Im guessing when i start building the backend i will have to use reasoning models more. 

Cursors tab autocomplete i think is the best tool for generating code tho. If i knew a bit about React or Typescript i wouldnt be using the chat or composer features at all. Knowing a bit about the language makes it easier to understand the code the LLMs generate and then make the appropriate changes. Therefore ive noticed that the code that the LLMs have been generating is not programmer friendly and comprehensive. What im currently doing is that when im done with a component i will be asking a reasoning model to denest the code using extraction and inversion techniques and also then i also add comments wherever i think it would be useful. At some point i will be studying the code more thorougly so i can learn more about the language and the framework. Right now im trying to see how far i can get by almost writing no code.

I can see that people using these AI assistants may be able to code some basic functionalities and some superficial concepts but i believe that as you get deeper into the project the AI will have a hard time coping with the complexity of the code. You will either have to be more specific and detailed with your promts or you must have a good understanding of the language and the framework. So at some point im guessing i will have to do manual work on this project.

Anyways the progress ive done in such a small amount of time is out of this world for me. Even if i knew how to write React and Typescript i wouldnt be nowhere near where i am now in 2 days.
